# Iris Dataset Classification - Supervised Learning

## Overview
Classification of Iris flowers into 3 species using supervised learning algorithms.

## Algorithms Used
- Linear Regression
- Logistic Regression
- KNN (K-Nearest Neighbors)

## Dataset
- Source: Kaggle (UCI Iris Dataset)
- 150 samples, 4 features
- 80% Training | 20% Testing

## Results
- Logistic Regression Accuracy: 100%
- KNN Accuracy: 100%

## Technologies
Python | Scikit-learn | Pandas | Matplotlib | Seaborn
